Lista de tarefas provisórias para projeto de conclusão:

  1. Pelo menos 3 páginas HTML
  2. Pelo menos 1 arquivo CSS de estilo e usar em todo o projeto
  3. Estrutura básica HTML com header, main, footer, section, nav e outros
  4. Parágrafo, quebra de linhas, negrito, itálico e sublinhado
  5. Menu de navegação dentro das TAGs nav
  6. Pelo menos uma imagem
  7. Link para o Linkedin pessoal e/ou GitHub
  8. Uma página "sobre" onde fala sobre seu projeto
  9. Lista ordenada
  10. Design responsivo usando Flexbox
  11. Personalizar um Favicon
  12. Adicionar um vídeo do Youtube incorporado
  13. Adicionar animação em CSS simples como eventos de mouse
  14. Pelo menos uma tabela
  15. Formulário
Essa lista é provisória e é postada com consentimento dos alunos.

Irei posteriormente colocar aqui o arquivo compactado para os alunos poderem baixar os seus códigos para estudos.

A lista de tarefas é provisória, no caso não está definido que será exatamente essas exigências para conclusão do curso.
